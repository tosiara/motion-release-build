# motion-release-build

Docker script to build multiple motion releases

# Supported platforms

* ubuntu:noble
* ubuntu:jammy
* ubuntu:xenial
* ubuntu:bionic
* ubuntu:focal
* debian:stretch
* debian:buster
* debian:bullseye
* debian:bookworm

# Supported archinectures
 
 * amd64
 * i386
 * arm64v8
 * arm32v7
 * arm32v5
  
 # Not supported
 
 * i386/ubuntu:focal - libmariadbclient-dev not available
